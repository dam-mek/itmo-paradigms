# Paradigms Homework (ITMO, Second Semester)

This repository contains homework assignments for the **Paradigms** course taken at **ITMO University** during the second semester.

## Project Structure

[![](https://img.shields.io/static/v1?style=for-the-badge&label=&message=Clojure&color=5881D8&logo=Clojure&logoColor=white)](https://github.com/dam-mek/paradigms/tree/main/clojure-solutions)
- **clojure-solutions**: solutions to problems in Clojure
  - *[expression](https://github.com/dam-mek/paradigms/blob/main/clojure-solutions/expression.clj)*: arithmetic expression parser
  - *[linear](https://github.com/dam-mek/paradigms/blob/main/clojure-solutions/linear.clj)*: code for linear mathematics


[![](https://img.shields.io/static/v1?style=for-the-badge&label=&message=javascript&color=F7DF1E&logo=javascript&logoColor=white)](https://github.com/dam-mek/paradigms/tree/main/javascript-solutions)
- **javascript-solutions**: solutions to problems in javascript
  - *[functionalExpression](https://github.com/dam-mek/paradigms/blob/main/javascript-solutions/functionalExpression.js)*: arithmetic expression parser
  - *[objectExpression](https://github.com/dam-mek/paradigms/blob/main/javascript-solutions/objectExpression.js)*: a cooler code for linear mathematics


[![](https://img.shields.io/static/v1?style=for-the-badge&label=&message=java&color=E11F21&logo=coffeescript&logoColor=white)](https://github.com/dam-mek/paradigms/tree/main/java-solutions)
- **java-solutions**: solutions to problems in java
  - *[expression](https://github.com/dam-mek/paradigms/tree/main/java-solutions/expression)*: arithmetic expression parser
  - *[queue](https://github.com/dam-mek/paradigms/tree/main/java-solutions/queue)*: implementation of LinkedQueue and ArrayQueue
  - *[search](https://github.com/dam-mek/paradigms/tree/main/java-solutions/search)*: iterative and recursive binary search in an array with contract-enforcing proofs in terms of Hoare triples


[![](https://img.shields.io/static/v1?style=for-the-badge&label=&message=prolog&color=FF8000&logo=coderwall&logoColor=white)](https://github.com/dam-mek/paradigms/tree/main/prolog-solutions)
- **prolog-solutions**: solutions to problems in Clojure
  - *[expression](https://github.com/dam-mek/paradigms/blob/main/prolog-solutions/expression.pl)*: arithmetic expression parser
  - *[primes](https://github.com/dam-mek/paradigms/blob/main/prolog-solutions/primes.pl)*: counting prime numbers
  - *[tree-map](https://github.com/dam-mek/paradigms/blob/main/prolog-solutions/tree-map.pl)*: implementation of AVL tree
